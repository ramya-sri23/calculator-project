Weather Forecast App

The Weather Forecast App is a responsive web application designed to provide real-time and accurate weather information for any location. The app fetches live data using the OpenWeather API and presents it in a clean, user-friendly interface. Users can search for any city to view current weather conditions, including temperature, humidity, wind speed, atmospheric pressure, and weather status. The application also displays sunrise and sunset timings, air quality index (AQI), and a 5-day weather forecast, helping users plan their activities effectively. The app is built using HTML, CSS (Bootstrap), and JavaScript, ensuring fast performance and cross-device compatibility. It uses API integration, asynchronous JavaScript (fetch API), and dynamic DOM manipulation to update weather data in real time. This project demonstrates practical skills in frontend development, API handling, data visualization, and responsive design, making it suitable for real-world use.

Technologies:HTML,CSS,JAVASCRIPT
